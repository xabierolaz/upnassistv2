# Sesión 1: Matemáticas

Python se comporta como una calculadora avanzada.

| Operador | Descripción |
|---|---|
| `+`, `-`, `*` | Básicos |
| `/` | División (siempre devuelve float) |
| `//` | División Entera (descarta decimales) |
| `%` | Módulo (Resto de la división) |
| `**` | Potencia |

---

### Misión: Ejercicios 11, 15 y 48 (Lab01)
1.  Declara `a = 10` y `b = 3`.
2.  Calcula e imprime la **suma** (`13`).
3.  Calcula e imprime el **módulo** (resto) de `a` entre `b` (`1`).
4.  **Círculo:** Declara `radio = 5`. Calcula el área (`3.14159 * radio ** 2`) e imprímela.
