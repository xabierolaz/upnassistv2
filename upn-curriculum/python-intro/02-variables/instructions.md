# Sesión 1: Variables

## Teoría
Una variable es un nombre que apunta a un valor almacenado en la memoria.
*   En Python, el tipado es **dinámico**: `x = 5` (entero) y luego `x = "texto"` (string) es válido.
*   Nombres descriptivos: Usa `snake_case` (ej. `mi_variable`).

## Tipos de Datos Básicos
*   `int`: Enteros (5, -10)
*   `float`: Reales (3.14, -0.01)
*   `str`: Cadenas de texto ("Hola")
