# Sesión 1: Lógica Booleana

## Teoría
El tipo `bool` solo tiene dos valores: `True` (Verdadero) y `False` (Falso).
Son fundamentales para tomar decisiones.

### Operadores de Comparación
*   `==` (Igual a)
*   `!=` (Diferente de)
*   `>` , `<` (Mayor/Menor que)

### Operadores Lógicos
*   `and`: Ambas deben ser True.
*   `or`: Al menos una debe ser True.
*   `not`: Invierte el valor.
