# Sesión 1: Introducción y `print()`

## Teoría: ¿Qué es Python?
Basado en *Sesión 1*:
Un ordenador no es inteligente; solo sigue órdenes. Python es un lenguaje **interpretado**, lo que significa que un programa llamado "intérprete" lee tu código línea por línea y lo ejecuta al vuelo.

A diferencia de los lenguajes compilados (como C o Pascal) donde debes traducir todo antes de ejecutar, Python es ideal para probar ideas rápidamente.

## La función `print()`
Sirve para mostrar información en la consola.
*   Texto: `print("Hola")` (Con comillas dobles o simples).
*   Números: `print(5)`
